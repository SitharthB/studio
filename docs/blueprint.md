# **App Name**: DocuQuery

## Core Features:

- Document Upload and Organization: Allow users to upload documents (PDF, TXT, etc.) and organize them into standalone files or collections.
- Collection Management: Enable users to create, rename, delete, and move documents between collections.
- Chat Interface: Provide a chat interface where users can ask questions related to the uploaded documents. Maintains a history of past chats.
- Document Selection: Allow users to select specific documents or collections to use as the context for their questions.
- Intelligent Answer Generation: Use a tool based on a Large Language Model to generate answers to user questions based on the selected documents.
- Cited Answer Display: Display the AI-generated answers with citations indicating the source documents and relevant passages.
- Citation Preview: On clicking the citation, display a side panel with the full document and highlight the cited text.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5), almost white, for a clean and uncluttered workspace.
- Accent color: Vivid orange (#FF9800) to draw attention to the most important interactive elements.
- Headline font: 'Space Grotesk', a proportional sans-serif, for a computerized techy scientific look; for headlines
- Body font: 'Inter', a grotesque-style sans-serif, for a modern machined neutral look; for body text.
- Use simple, professional icons for document types, collections, and actions.
- Maintain a clean, two-pane layout with a sidebar for document management and a main content area for questions and answers.
